:::info Note
This library relies on Expo Modules Code, and thus requires Expo to be installed.
:::

### Add the package

```console
yarn add swiftui-react-native
```

### Prebuild and Run

```console
npx expo prebuild -p ios
```

```console
npx expo run:ios
```

### That's it!

You should be all set now. Go build something awesome!
